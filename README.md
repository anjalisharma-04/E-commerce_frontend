# ShopMart – Smart E-Commerce Platform

ShopMart is a modern, responsive e-commerce frontend built using React, Vite, and Tailwind CSS.  
The project follows a clean, scalable folder structure and implements core shopping features.

---

##  Tech Stack
- React (Vite)
- Tailwind CSS
- React Router DOM
- Context API (Cart)
- Lucide Icons

---

##  Folder Structure
src/
 assets/          # Images & static files
 components/      # Reusable UI components
 pages/           # Route-based pages
 layouts/         # App layout (Header/Footer)
 routes/          # App routing
 data/            # Dummy data
 services/        # Context & services
 utils/           # Helpers

---

##  Features
- Home page with hero banner
- Category browsing
- Featured products
- Product details page
- Add to cart functionality
- Cart page with live count
- Order tracking UI
- Login & Register pages
- Fully responsive design

---

##  How to Run
npm install  
npm run dev  

---

##  Note
This is a frontend-only project. Backend/API can be integrated later.
